# Проект: Путешествие по России

### **Описание:**

Проект адаптивной веб-страницы, подстраивающейся под устройства с различными разрешениями экранов.

### **Проект выполнен с применением:**

- БЭМ
- Медиа-запросы
- Flexbox
- Grid-layouts
